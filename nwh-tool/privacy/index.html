NWH Tool — Privacy Policy

Last updated: May 19, 2026

Operator: Pacific Homeworks Northwest LLC (Tacoma, WA, USA)
Contact: tim@nwhomeworks.com

Scope

NWH Tool is a private, internal application built and operated by Pacific Homeworks Northwest LLC for the sole purpose of managing the company's own job costing, cash allocation, and bookkeeping. It is not distributed to the public, listed in any app marketplace, or made available to other businesses. This policy describes how data is handled when the application is used by Pacific Homeworks Northwest LLC personnel.

Data the application accesses

When connected to QuickBooks Online, NWH Tool requests the com.intuit.quickbooks.accounting scope. Within that scope, it reads accounting records associated with Pacific Homeworks Northwest LLC's own QuickBooks company — including customers, vendors, invoices, bills, payments, accounts, items, and journal entries. The application may also propose write operations (creating or updating these same record types) for review and explicit approval by an authorized user before any write reaches QuickBooks.

The application does not access QuickBooks records belonging to any other company or person, and does not access QuickBooks Payments or any other Intuit product.

Where data is stored

All data accessed from QuickBooks, all OAuth tokens (access and refresh), and all application-generated records are stored locally on devices owned and controlled by Pacific Homeworks Northwest LLC. Storage takes the form of a local SQLite database file on the operator's workstation. No application data is transmitted to or stored on third-party servers, cloud services, or hosted databases.

Sharing

NWH Tool does not share, sell, lease, license, or otherwise transfer any data to third parties. Data flows only between the QuickBooks Online API and the local application instance authorized by Pacific Homeworks Northwest LLC.

Retention

Data is retained locally for as long as it is useful to Pacific Homeworks Northwest LLC's bookkeeping needs. Authorized users may delete the local database at any time, which removes all stored records and revokes the operator's local copy of OAuth tokens.

Revoking access

Authorized users may disconnect NWH Tool from QuickBooks Online at any time by either (a) using the in-app Disconnect control, or (b) signing in to QuickBooks Online, opening Settings → Apps, locating "NWH Tool," and selecting Disconnect. Disconnection invalidates the application's OAuth tokens and ends its ability to read or write QuickBooks data.

Security

OAuth tokens are stored in a local database file protected by the operating system's file permissions on the operator's workstation. The application never logs, transmits, or displays the client secret. All communication with the QuickBooks Online API uses HTTPS. The application enforces an approval step before any write to QuickBooks is executed.

Children

NWH Tool is a business application and is not directed at children. It does not knowingly collect data from anyone under 13.

Changes to this policy

If this policy is updated, the "Last updated" date above will be revised and the new version will be published at the same URL.

Contact

Questions about this policy may be directed to tim@nwhomeworks.com.
