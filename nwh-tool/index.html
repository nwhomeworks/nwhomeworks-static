NWH Tool

A private internal application for Pacific Homeworks Northwest LLC.

What it is

NWH Tool is a job-costing and cash-allocation application built by and for Pacific Homeworks Northwest LLC (DBA NW Homeworks) for internal bookkeeping use only. It connects to the company's own QuickBooks Online account to read and (with explicit operator approval) write accounting records related to active remodeling projects.

The application is not distributed to the public, not listed in any app marketplace, and is not available to other companies.

How authorized users interact with it

Launching the application

Authorized personnel run the application locally on a company-controlled workstation. The application starts a local server bound to localhost on the user's machine; access is from a browser on the same machine. There is no public web interface for the application.

Connecting to QuickBooks Online

From within the local application dashboard, an authorized user clicks Connect to QuickBooks. This initiates the standard Intuit OAuth 2.0 authorization-code flow. The user authenticates with their QuickBooks credentials, approves the requested com.intuit.quickbooks.accounting scope, and Intuit redirects back to the local application, which stores the resulting OAuth tokens in a local database file on the operator's workstation.

Reconnecting (after a prior disconnect, or after the refresh-token window expires) follows the same flow.

Disconnecting from QuickBooks Online

Authorized users may disconnect in either of two ways:

  1. From within the application — the dashboard exposes a Disconnect control that revokes the local copy of the OAuth tokens.
  2. From QuickBooks Online directly — sign in to QuickBooks Online, open Settings → Apps, locate "NWH Tool," and select Disconnect.

Either method invalidates the application's access and ends its ability to read or write QuickBooks data.
